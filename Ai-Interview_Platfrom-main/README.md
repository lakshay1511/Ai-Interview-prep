# PrepShape ‚Äì AI-Powered Interview Preparation Platform

PrepShape is an AI-driven interview practice and analysis platform designed to help candidates improve their technical and behavioral interview skills. It provides **real-time mock interviews**, **performance analysis**, **personalized feedback**, and **domain-specific interview generation** based on experience level and tech stack.

Built with **Next.js**, **Firebase**, and **shadcn/ui**, PrepShape delivers a modern, fast, and responsive user experience.

## üöÄ Features

### **1. AI-Powered Mock Interview**
- Start a full AI-conducted interview session.
- The bot asks questions, listens to your responses (video/audio), and analyzes them.
- Get score-based evaluation on:
  - Communication clarity  
  - Technical correctness  
  - Confidence  
  - Completeness of answer  
  - Overall performance indicators
- Receive structured feedback and improvement suggestions.

### **2. Custom Interview Generation**
You can create your own interview session based on:

- **Experience Level**:  
  - Beginner  
  - Intermediate  
  - Experienced  

- **Tech Stack**:  
  Choose from various technologies (e.g., React, Node.js, Python, ML, DBMS, DSA, etc.)

- **Projects You Have Built**:  
  The bot generates personalized questions based on your submitted project details.

### **3. Practice Anytime**
- Unlimited practice with newly generated question sets.
- Auto-generated follow-up questions for deeper evaluation.
- Clean, minimal, and responsive UI powered by **shadcn components**.
